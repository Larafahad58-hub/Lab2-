# Lab2-

House Prices Dataset Summary 
	•	The dataset contains information on real estate properties, including various features affecting the price.
	•	It consists of 18 columns representing different attributes such as price, bedrooms, bathrooms, square footage, and more.
	•	The key features include:
	1.	Price: The sale price of the house.
	2.	Bedrooms: The number of bedrooms.
	3.	Bathrooms: The number of bathrooms.
	4.	Living area (sqft_living) and total land area (sqft_lot).
	5.	Waterfront, view, and condition of the property.
	6.	Number of floors and sqft_above (upper floors area).
	•	The machine learning problem is to predict house prices using regression models.
	•	Steps in the project:
	1.	Load and clean the dataset.
	2.	Explore data and analyze the relationships between features.
	3.	Build a regression model (e.g., Linear Regression).
	4.	Evaluate and improve the model.
	5.	Use the model to predict house prices.
